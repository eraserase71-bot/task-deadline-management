# 대학생 과제 마감 관리 웹앱

## 프로젝트 소개

이 웹앱은 대학생이 여러 수업의 과제, 발표, 시험 일정을 한눈에 정리하고 마감일까지 남은 날짜를 확인할 수 있도록 만든 과제 관리 프로그램이다.

## 주요 기능

* 과제 등록
* 수업명, 과제명, 마감일 입력
* 중요도 선택
* 진행 상태 선택
* D-day 자동 계산
* 과제 목록 카드 표시
* 과제 수정
* 과제 삭제
* localStorage를 이용한 데이터 저장

## 사용 방법

1. 수업명과 과제명을 입력한다.
2. 마감일, 중요도, 진행 상태를 선택한다.
3. 저장 버튼을 누른다.
4. 과제 목록에서 D-day와 상태를 확인한다.
5. 필요하면 수정 또는 삭제한다.

## 검증 내용

* 빈칸 입력 시 저장되지 않는지 확인
* 과제 추가 후 새로고침해도 데이터가 유지되는지 확인
* 수정/삭제 버튼이 정상 작동하는지 확인
* 모바일 화면에서도 레이아웃이 깨지지 않는지 확인
